# SGBP - Sistema de Gerenciamento de Biblioteca (UMC)

Este projeto faz parte da Etapa 3 da disciplina de Análise e Projeto de Softeare na UMC.

## 🚀 Tecnologias
* Java 21
* Spring Boot 3.4
* MongoDB Atlas (Cloud)
* Thymeleaf & Tailwind CSS

## 🛠️ Como rodar o projeto
1. Clone o repositório.
2. Importe como um projeto Maven no IntelliJ.
3. Configure sua URI do MongoDB no arquivo `application.properties`.
4. Execute a classe `SgbpBibliotecaApplication`.
